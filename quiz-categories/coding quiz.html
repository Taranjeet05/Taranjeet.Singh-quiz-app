Coding Quiz

1. Which HTML element is used to define the main heading of a webpage?

Answer: <h1>

2. Which CSS property is used to change the text color of an element?

Answer: color

3. Which JavaScript method is used to select an element by its ID?

Answer: getElementById

4. Which attribute is used to define inline styles in HTML?

Answer: style

5. Which CSS property is used to make text italic?

Answer: font-style

6. What is the default value of the CSS position property?

Answer: static

7. Which JavaScript keyword is used to declare a variable?

Answer: var

8. Which HTML tag is used to display a numbered list?

Answer: <ol>

9. Which CSS property controls the size of text?

Answer: font-size

10. Which JavaScript function is used to display a message in a dialog box?

Answer: alert
